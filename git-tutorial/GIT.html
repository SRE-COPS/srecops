Git Tutorial

A brief description of what this project does and who it’s for ## Installing Git

You can download and install Git from the official website: Git Downloads

Configuring Git

After installation, configure your name and email address. These will be associated with your commits.

$ git config --global user.name "Your Name"
$ git config --global user.email "youremail@example.com"

Creating a New Git Repository

To start version controlling a project, initialize a Git repository in the project directory.

$ git init

Staging and Committing Changes

Stage your changes for commit:

$ git add file.txt  # Stage a specific file
$ git add .         # Stage all changes

Commit your changes:

git commit -m "Your commit message here"

Checking the Status

Check the status of your repository to see what changes are staged and what’s not.

$ git status

Viewing Commit History

View the commit history of your repository.

$ git log

Working with Branches

Create a new branch:

$ git branch new-feature

Switch to a branch:

$ git checkout new-feature

Merge a branch:

$ git merge branch-to-merge

Remote Repositories

Add a remote repository:

$ git remote add origin https://github.com/yourusername/your-repo.git

Push changes to a remote repository:

$ git push -u origin main

Pull changes from a remote repository:

$ git pull origin main
